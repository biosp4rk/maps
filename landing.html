Filtering

  • Filters are case insensitive
  • Search for addresses by putting = before a hex address
    • =3897C - entry with the address 0x3897C
  • Search for address ranges by putting >, <, >=, or <= before a hex address
    • <=A8D3C - entries with an address of 0xA8D3C or less
    • >=42D74 <43788 - entries with an address between 0x42D74 and 0x43788
  • Search for entries near an address by putting ~ before a hex address
    • If the address is within an entry, it returns the matched entry and the entries before and after
      Otherwise, it returns the nearest entries before and after the address
    • ~43788 - entry at 0x43788 (if an entry matches) and the entries before and after
  • Search for phrases by putting " around the phrase
    • "chozo statue" - entries with the phrase "chozo statue"
  • Search for regex patterns by putting / around the pattern
    • /sprites?/ - entries with "sprite" or "sprites"
    • /(ridley|kraid)/ statue - entries with "ridley statue" or "kraid statue"
  • Put - before a term, phrase, or pattern to exclude it from the results
    • samus -eater - entries with "samus" that don't have "eater"
  • Clear a filter by pressing esc
